생성자 : 클래스의 인스턴스를 만들어주는 함수
-생성자의 이름은 클래스의 이름과 같다.

ex)Student s1 = new Student(    );
- 괄호가 비어 있다면 기본생성자라고 함.

생성자는 반환타입을 적을 수 없음.

Student(     ){  - 기본생성자를 정의, 컴파일 할때 자동으로 만들어줘서 실제로 적진 않음
}

ex)
생성자를 직접만드는 경우
Student(int n, String c, char s){  만드는 클래스에 입력
}

Student s1 = new Student(1, "홍길동", 'M'); - 메인클래스에 입력

-생성자를 직접만들어서 사용하는 경우 기본 생성자를 사용자가 직접 넣어줘야한다.


생성자의 정의: 생성자는 해당 클래스의 인스턴스가 만들어질때
클래스의 멤버변수를 초기화하는데 사용된다 

- 함수는 이름만으로 구분하지 않음.
변수와 달리 함수들을 함수를 구분 짓는데에는 함수의 시그니처(매개변수, 함수의 이름, 함수의타입)가 사용된다.
